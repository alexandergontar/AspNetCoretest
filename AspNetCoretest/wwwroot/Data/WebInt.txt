The Web Interface.
It allows remote control through a Web browser of a PC, Laptop or a mobile device.
The initial connection to a local Wi-Fi network is also established through it.
Для этого необходимо включить устройство,
Подключиться к нему в режиме точки доступа (название сети будет начинаться с ESP8266..)
Пароль находится в исходном С++ коде в описании, строка 94:
 WiFi.softAP("ESP8266", "yourpassword");
После подключения, в адресной строке браузера набрать: 192.168.4.1
появится веб-интерфейс управления(см. фото). 
Для подключения к местной Wi-Fi сети  нажмите кнопку Wi-Fi options, введите
имя и пароль вашей сети, нажмите Save. После этого, надо подождать несколько секунд, пока
обновляется веб страница. Затем в случае удачногого соединения появится IP-адрес подключения
к локальной сети, либо код ошибки подключения. После этого можно отключиться от точки доступа и зайти
на устройстово уже из локльной сети по указанному адресу подключения.
