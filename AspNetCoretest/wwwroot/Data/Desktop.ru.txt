Десктоп.
Выше можно видеть изображение десктоп-приложения для удаленного
управления устройством. Кнопка Monitor/Stop используется для включения/выключения непрерывного мониторинга.
Интерфейс приложения объясняет сам себя. Кнопка Wi-Fi в этом проекте не используется.
Проект этого приложения в Microsoft Visual Studio можно найти по следующей ссылке:
https://github.com/alexandergontar/ClientSocket/tree/main/ClientSocket/ClientSocket
либо нажать на синий значок ">>>>>>>>>>>>>" внизу.
Те, кто не хотят редактировать проект в Microsoft Visual Studio могут сразу зайти в папку Release
и запустить приложение двойным кликом файла ClientSocket.exe.
